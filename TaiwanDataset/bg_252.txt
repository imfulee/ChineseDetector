: 身為前農民，小弟來分享一下所見所聞 : 1. 收入 : 收入幾百萬都騙人的啦，最好農民收入都有幾百萬 : 個人年收有破百的佔總農民人數應該不到5% : 多的是全家五六個人一起工作，然後總收入不到100萬 : 個人年收入30萬內的佔絕大多數 : 因為種田收入低，鄉下很多地都廢耕 : 然後種一種綠色的草，每年領取幾千塊補助 : 現在可以務農到年收破百的，大多是鍵盤農民 : 2. 很累 : 務農沒有假期，你一塊地種東西就要每天去巡 : 巡一下病蟲害，巡一下有沒有人來偷 : 病蟲害跟有人偷都可以讓你一次損失10幾萬 : 採收的時候要連續幾天在大太陽底下採收 : 你請一個人幫忙就等於要多花兩三千元 : 這對於一般農家負擔很大 : 只有等收成了，要種下個作物之前可以稍微休息幾天 : 3. 很髒 : 我知道說出務農很髒會被道德魔人噓 : 但是我爸媽都被鄰居小孩嫌過髒臭 : 務農的相對上班族比較髒是事實，噓我也沒用 : 4. 住鄉下 : 沒地方逛，頂多鄉下小夜市 : 沒有休閒，就只能滑手機 : 5. 子女教育 : 鄉下教育資源相對缺乏 : 小孩很少可以讀到明星學校 : 另外可能是因為階級複製 : 我家鄉務農的人子女很少脫離務農 : 很多人都是做其他工作不上不下 : 只好回去住家裡，順便幫忙 : 每個月領一兩萬的錢，吃不飽餓不死 : 6. 菸、酒、檳榔、阿B不離身 : 很多農民都有這些嗜好 : 老了很多慢性病纏身 : 以上是我看到屏東農村的務農現況 : 大部分農家都是過著貧窮的生活 : 請好好想想，如果每個務農的都可以年收百萬 : 那為什麼這麼多年輕人要去澳洲打工？ : 為什麼那麼多年輕人要做外送？ : 務農是一條很難走的路，幸好我脫離了這條路 : 只是偶爾想想以前家裡過的生活，還是覺得很慘。 在文章下認真回覆你，反而被當鍵盤俠？ 小弟老家在屏東雖未多麼大富大貴，但至少老一輩的靠種植稻米、蓮霧以及養殖魚蝦能讓 我們過上還不錯的生活。 另外小弟的外公剛好是台灣在1996年第一次成立有機米產銷班的班長，在當時外公決定研 發種植有機米時，周圍的人都不看好，但憑藉自身的堅持和努力，有機米現在已是米食市 場的一個選擇。 且栽種有機米的期間也同步學習秧苗的培育，慢慢成為秧苗的供應商，今年寒假回去拜年 時得知育苗場已達一期可提供幾十萬箱秧苗產能給花東地區的農田耕作。 我內外兩個祖父都是從一介小農民開始做起，但為什麼他們可以靠務農有不錯的收入，而 你只會對務農這事怨聲載道？是他們運氣好年年豐收嗎？還是他們在自己的工作上很努力 ？ 對了，還有你嘴了我的收入跟那麼好賺怎麼不去務農？ 我確實沒有去務農繼承家業，因為我覺得工作是自己的一個選擇啊，為什麼我要去做一個 自己沒有興趣的工作？ 至於我目前的收入，小弟我還在大學就讀中，但未拿取家中資源有經營自己的小生意，上 個月淨利潤有8左右，還沒到你年收百萬的標準真是慚愧。 當然，你也可以覺得我整篇都在唬爛，當作一個死屁孩被激到亂發的幻想文就好。 。推一個選工作不是只看錢，心態正確做什麼都會有成就～。看完文章就知道有些人會失敗不是沒有原因的。@@務農能賺大錢的真的是少數阿，這篇文章不也是算是特例？。在農村過上不錯的生活是因為很多生活成本都不用負擔
