我覺得你們應該會分手，但問題不在你。 我以一個觀察躁鬱症患者的經驗來跟大家分享我觀察到的東西。 首先我必須說，躁鬱症不是這些人自願想要得的病，這不是他們的錯。 確實躁鬱症的病人不會接受自己有這種病徵，但他們發病其實一開始他們分辨不出來。 加上社會給這些人貼上標籤所以她們會抗拒就醫，尤其是家人朋友的態度是”你有病要去 看醫生”，這很容易引起他們的反抗心反而不容易善後。 其實躁鬱症病患跟一般人一樣，他們並不希望自己生病，一旦身旁的朋友家人發現她們開 始亂買東西，晚上睡不著，緊張性頭痛，亢奮，這可能就是燥期初期，這時間趕快就醫其 實很容易得到控制。 我的經驗其實是先建立病患“就醫不是你有病”的概念，燥期其實很容易發現，也很容易 被治療。預防性治療其實是避免惡化的關鍵，現在針對躁鬱症有長效性針劑，只要醫生評 估患者可以施打就可以有半年的藥效，定期追蹤就可以跟正常人一樣生活。 躁鬱症嚴重時，例如攻擊人，砸東西可以報警強制就醫。這時候大概就是需要住院34週 ，這是必要的措施所以平常心看待就好。 說實在對這類病人，最好的相處方式就是，不要因為他們生病就改變對他們的眼光，責備 ，輕視，逃避是不能改變問題，唯獨耐心陪他們建立屬於他們的自救方式才是最重要的。 我們要讓他們了解，定期看醫生是確保他們可以繼續享受人生快樂時光唯一的方式。 然而每個人生來就是為了挑戰而來，不要因為身心問題而氣餒，這是有藥醫，不是沒藥醫 。 至於病患身邊的家屬朋友，我覺得保持平常心很重要，因為病患進入比較嚴重的狀態時其 實他們的言語是很容易激怒人的，初期如果沒有控制住，各位能做的就是強制送醫，進去 裡面住上一陣子。 出院後再回診請醫生打個半年針觀察一個月就會回到原本的樣子。 這時候我希望身邊的人仍舊還是平常心，迎接他回家，畢竟生病不是他們願意的，只是需 要大家耐心跟他們一起找方法。 每個人都希望自己是被用心對待，這類病患更需要，與其讓他們沉淪造成更多問題，不如 陪著他建立一個屬於他自己的自救機制，既然可以在醫療制度下當個正常人，為何還要當 個鬼？當個失去自由的人？ : 如題，文長慎入，手機排版傷眼見諒。先說個人背景，本身有過動症， : 也有憂鬱病史，但目前已恢復到不影響生活的程度，目前研究所備考中。 : 與女友交往三年多，同居兩年餘。 : 雙方學歷有點差異，我中字科科相關，她私立文組延畢。 : 在一起後本來也沒什麼大問題，女友本身有輕微的焦慮傾向跟社交恐懼， : 但都影響不大，直到半年前她和她一直很敬愛的姐姐大打出手鬧翻， : 從此她的人際就陷入了惡性循環，不停的害怕身邊的人拋棄她，因此一直將朋友推開， : 做出各種言語傷害或情緒勒索朋友的行為，當然我更是首當其衝。 : 也開始了許多難以控制的行為，比如三個月內將戶頭十幾萬揮霍殆盡、 躁鬱症的燥期特性 : 想方設法要報復受不了她而離去的朋友、無限擴張自己的自卑而對他人動輒得咎、 : 逃避為自己的行為負責任、對未來規劃朝三暮四、虎頭蛇尾， : 等族繁不及備載之異於常人的行為舉止。 : 雖然有持續在就醫並諮商，但狀況依舊時好時壞， : 大四結束後因為英文門檻未過遲遲無法畢業，也拒絕念書考多益，就這樣無業了一年多 : 。 : 最近她透過朋友的關係找了份服務業的兼差、也在流浪動物相關組織做志工， : 本以為病情稍有好轉了，就在昨晚下班時，因為帶她的同事前輩催促她加快收拾速度， : 她就突然發作，拿出包裡的美工刀割自己的手臂，當場嚇到介紹她工作的朋友及同事， : 後續更是直接衝到馬路中央坐下，直到路人協助報警送醫， : 我趕到醫院時恰好遇到接獲警方通知而從外縣市趕來的她父母。 : 我與女友的幾位朋友商討，大家一致認為女友應強制住院或回家交由父母照顧， : 但她在急診室極度不配合，拒絕任何醫療行為、也拒絕隨父母回家。 : 折騰到快3點才終於回到租屋處，女友期間也是不斷抱怨工作環境、 : 甚至怪罪介紹工作給她的朋友，還報復性的將傷口的血腥照片傳給主管。 : 女友與她朋友相識十幾年，而只因為朋友混亂中遺失女友的健保卡而堅持與對方決裂， : 完全不願面對朋友受驚嚇卻還在急診室陪伴女友到2點多才回家。 : 看到女友手上滿佈的傷口跟想起所有的一切，我現在心裡對女友產生很深的恐懼， : 我很想分手回老家，但路途實在遙遠，且瀕臨考期，完全沒心力搬家， : 而老家與考場分別在南北兩端，不可能有精力連續數天搭長途車趕考， : 可是女友這樣的精神狀態，我也很擔心我養的三隻貓的安危。 : 所以現在要5點了還在這裡發文，懇求男女板板友們給我一些建議，拜託大家了。 :
