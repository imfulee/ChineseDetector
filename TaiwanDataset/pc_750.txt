AMD的Ryzen 3 1200 CPU最近收到了新修訂，已更新為Zen 12nm核心架構，稱為Ryzen 3 1200 AF 此更新的架構沒有提供更快的時賣速度 而是提供更穩定的時賣和更高的超頻應用於該處理器，並且還提供了更好的記憶體超頻。 AMD Ryzen 3 1200 AF處理器升級到12 nm Zen架構。與原始Ryzen 3 1200相比，該處理 器的主要特點是更高的超頻 另一個特點是更好的記憶體控制器，它可以實現更快的記憶體超頻。如GameER的Youtube 影片所示 此經過更新的處理器的頻率最高可達4.0 GHz，這是經過大幅改進的架構所推動的 在GameER的影片中，它展示了即使沒有4.0 GHz超頻，他仍能夠比AF型號上的原始Ryzen 3 1200 CPU更快地獲得性能 因為它在大多數測試中仍勝過Intel Core i57400。 GameER通過特定測試執行這些處理器，包括Cinebench 20，Cinebench 15多線程， Cinebench 15單線程 與原始Ryzen 3 1200，Intel Core i57400和Intel Core i39100F相比，這三個不同的 測試將顯示Ryzen 3 1200超出的地方。 超頻的Ryzen 3 1200 AF和Ryzen 3 1200 AF（非超頻）相比，Cinebench 20第一名仍是 Intel Core i39100F 更新的Ryzen排名第二，超頻得分為1590，非超頻得分為1368，而i39100F得分為1640 這意味著超頻的Ryzen能夠接近I39100F處理器在100 pt之內。 其他兩個基準測試（Cinebench 15多線程和Cinebench 15單線程）都向我們展示了一個不 同的故事 超頻的Ryzen在多線程基準測試中領先大約50分，在單線程測試中領先2分。非超頻版本緊 隨i39100F處理器 在多線程方面落後40 pt，在單線程基準方面落後20 pt。 該晶片的遊戲性能對於預算四核晶片也非常出色。當然對於那些預算有限的個人電腦肯定 會選擇Ryzen 3 1200 AF CPU 但是新的售價為99美元的Ryzen 3 3100也是一個很好的選擇，它可以通過新的核心架構進 一步擴展並提供更好的超頻性能結果。 Ryzen 3 1200 AF的性能在經濟型PC主機中非常出色，功耗更低，超頻能力更好。它也帶 有盒裝散熱器 如果與A320甚至B450板搭配使用，也使其成為預算遊戲設置的理想合作夥伴。 來源 https://wccftech.com/amdryzen31200af12nmzenquadcorecputestedexcellentbudgetchip/ XF編譯 https://www.xfastest.com/thread24072411.html 香 過氣低價產品 但鬼島不賣 喜歡可以去美國買 。鬼島有3500x，這貨應該很難賣吧。有錯字 時賣(X)。像機翻的文章。面向低價市場卻沒內顯~ＱＱ
