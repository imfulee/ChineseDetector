Windows 95問世25年 文/林妍溱 20200825發表 . 8月24日似乎不是什麼特別日子，但在電腦史上卻有相當意義，因為是Windows 95上市25 周年紀念日，象徵現代化Windows作業系統重要版本已經跨過1/4世紀。 1995年8月24日，微軟共同創辦人之一比爾蓋茲宣布開賣Windows 95，為了促銷，微軟砸 下巨資舖天蓋地播放搭配滾石合唱團《Start Me Up》及狂打開始功能表的電視廣告 ，還請來《六人行》演員演出介紹Windows 95的情境短劇。微軟強力行銷下，Windows 95 頭5個星期就賣出700萬套，包括許多還沒有電腦的消費者。 Windows 95工程版本是Windows 4.0，最大的貢獻之一在於圖形使用者介面較Windows 3.1 的大幅革新。Windows 95某些元素一直沿用並保留在Windows 10中，包括開始功能表、工 作列、捷徑、Outlook、檔案總管、資源回收筒，以及視窗右上角關關或縮小、放大的按 鍵等。也是從Windows 95開始，使用者可以為檔案取長檔名，而不限於之前的8個字母。 Windows 95的開機音樂及視窗飄揚的開機畫面，開啟許多學生和上班人士的一天。 Windows 95種種設計讓一般用戶也能輕鬆使用，讓電腦不再只是程式高手的專利，奠定了 微軟在桌機電腦作業系統的獨大地位。 Windows 95也是第一個搭售微軟自有瀏覽器Internet Explorer（IE）的作業系統。這個 策略讓IE成功打敗原有最受歡迎的瀏覽器Netscape，一度市占率高達9成，也種下2000年 美國司法部對微軟的反托拉斯官司的主因。 25年後的今天，微軟不再獨霸運算世界。許多人用智慧型手機的時間超過電腦，曾經獨大 的IE今天幾乎瀕危，還好微軟及時改用Chromium為核心，才使後繼者Edge免於絕跡命運， 但是Bing搜尋早已被Google邊緣化。但Windows還是繼續保留住消費者，包括加入防毒及 其他安全功能、每月安全更新讓電腦更安全更穩定，當然另一原因是2015年推出的 Windows 10起，改成一年改版2次的軟體即服務，使用者不想升級也很難。在 Windows
