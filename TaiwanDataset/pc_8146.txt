MSI 最近發布了 AMD 500 系列主板新版 BIOS，主要是支援 AMD 新一代 5000 系列處理 器搭配 RX 6000 系列顯卡的 SMART ACCESS MEMORY 功能，讓玩家可以馬上體驗到 3A 平台效能的套裝加成。 AMD 的 SMART ACCESS MEMORY 功能是 AMD 借助新一代 PCIe 4.0 高速帶寬的新技術 突破了以往 Windows 系統 PC 一次只能使用 256MB 顯示卡記憶體的限制，消除了過去的 傳輸瓶頸 讓 CPU 和 GPU 之間的數據傳輸效率可以大大的提升，充分發揮 GPU 記憶體的潛力以提 高效能 讓玩家有更好的遊戲體驗。根據 AMD 官方的資料，通過打開這個功能，就能得到平均 6% 左右的提升。 在 MSI 主板上啟用 SMART ACCESS MEMORY 可以參考下方說明。 更新 BIOS 後，找到“Settings > Advanced > PCI Subsystem Settings” 啟用“Resize BAR”和“Above 4G memory / Crypto Currency mining”，按“ F10” ，保存設置並重新啟動。 SMART ACCESS MEMORY 這項功能必須要使用 AMD Radeon RX 6000 系列顯卡和AMD Ryzen 5000系列處理器 顯卡驅動也要更新到最新版本。完成這些操作步驟後，可以體驗到這項功能了。 目前 MSI 的所有 AMD 500 系列主板都可以使用此項新功能 用戶可以透過微星官網的相應產品支援頁面，下載最新的 BIOS 版本。 滄者編譯 https://www.coolaler.com/threads/msiamd500betabiossam.361482/ 是否會陷入改BIOS扣850的問題 。看你要SAM還是850。扣850是什麼！？。感謝林董debug。先扣850。主機板還好吧  沒有地方可以收850的bios費。主機板BIOS本來就可以刷啦。主板是doglike被嘴。技嘉的板也有了。以後主機板刷BIOS說不定也要收850。智力測驗 誰買誰__。所以刷下去會被七折嗎？。目前是沒聽過刷MB BIOS有被收過850的吧。板廠不都在BIOS頁面寫：不建議自行更新BIOS嗎？。M$I：不建議換還自己換，收你850剛好啦！。M$I: checksum不對 要收850刷回來。刷了以後送修會不會收 850 啊？。人家板廠上面就有說換BIOS就可以收850。爽賺850。先收850$再打七折，水啦！。就剩你400系還不放。想到之前版友送修godlike被換成他媽好客，還敢買啊。刷了會破保送修收850嗎?。搞臭了有沒有用過都先噴再說。七折，我還是買別家。450是不是被放生了啊。微星臭死了,商譽商譽阿!
